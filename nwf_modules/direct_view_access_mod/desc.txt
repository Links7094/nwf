Installation this module supports direct access to the views via file path for front-end debugging.
It is only enabled when the nwf.config.echoDebugInfo configuration is set to true.
E.g:
http://localhost:8099/test.html  --> www/view/test.html
http://localhost:8099/xx/test.html  --> www/view/xx/test.html
